# clean-scheduling

Customers select a cleaning service type from options such as move-in/move-out cleaning, office cleaning, commercial cleaning, specialized cleaning, or appliance cleaning. They input details like date, service type, address, and area size into the reservation form. This information is then displayed on a card screen, allowing the cleaning manager to review and submit a quote
